Text Input

  • type="text"
  • attributes: placeholder, maxlength, required
  • syntax: <input type="text" placeholder="Enter name">

Password Input

  • type="password"
  • attributes: minlength, maxlength, required
  • syntax: <input type="password" placeholder="Enter password">

Email Input

  • type="email"
  • attributes: required, multiple
  • syntax: <input type="email" placeholder="Enter email">

Number Input

  • type="number"
  • attributes: min, max, step
  • syntax: <input type="number" min="1" max="10">

Date Input

  • type="date"
  • attributes: min, max, value
  • syntax: <input type="date">

Checkbox

  • type="checkbox"
  • attributes: checked, value
  • syntax: <input type="checkbox" checked>
  • Note1: You have to check for each boxes/button individually whether they are clicked or not.
  • Note2: Name should be same if it is in group

Radio Button

  • type="radio"
  • attributes: name, value, checked
  • syntax: <input type="radio" name="gender">
  • Note1: You have to check for each boxes/button individually whether they are clicked or not.
  • Note2: Name should be same if it is in group

File Upload

  • type="file"
  • attributes: accept, multiple
  • syntax: <input type="file" accept="image/*">
  • Accept: "images/*" , "application/*","audio/*","video/*"
  • Accept: ".html,.js,.css"

Color Picker

  • type="color"
  • attributes: value
  • syntax: <input type="color">

Range Slider

  • type="range"
  • attributes: min, max, step, value
  • syntax: <input type="range" min="0" max="100">

Submit Button

  • type="submit"
  • attributes: value
  • syntax: <input type="submit" value="Submit">
