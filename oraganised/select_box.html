Select Box

  • element: <select> with <option> children
  • attributes: name, id, multiple, size, required
  • Syntax : <select name="options"> <option>...</option> </select>

Select Box with Multiple Selection

  • attribute: multiple
  • allows selecting more than one option
  • Syntax : <select name="items" multiple> ... </select>

Select Box with Default Selected Option

  • attribute: selected
  • pre-selects an option when the page loads
  • Syntax : <option value="x" selected> ... </option>
