Border

  • syntax: border: width style color;
  • border-style: solid, dashed, dotted, double, groove, ridge, inset, outset
  • border-width: thin, medium, thick, or specific values (px, em, etc.)
  • border-color: any valid CSS color (name, hex, rgb, etc.)
  • This text has a solid red border (2px)
  • This text has a dashed blue border (3px)
  • This text has a dotted green border (4px)

Border-radius

  • values: length units (px, em, rem), percentage (%)
  • Syntax : border-radius: value;
  • can be applied to each corner: border-top-left-radius, border-top-right-radius, border-bottom-left-radius, border-bottom-right-radius

This div has fully circular corners (50%)

Border-collapse

  • values: collapse, separate
  • Syntax : border-collapse: value;
  • applies only to table elements
