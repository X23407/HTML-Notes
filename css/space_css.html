Margin

  • values: length units (px, em, rem), percentage (%)
  • Syntax : margin: value;
  • can be applied to each side: margin-top, margin-right, margin-bottom, margin-left
This div has margin: 20px (space outside the border)
This div has margin: 10% (relative to parent width)
This div has margin only on top (30px) and left (50px)

Padding

  • values: length units (px, em, rem), percentage (%)
  • Syntax : padding: value;
  • can be applied to each side: padding-top, padding-right, padding-bottom, padding-left
This div has padding: 20px (space inside the border)
This div has padding: 5% (relative to parent width)
This div has padding only on top (30px) and left (50px)
